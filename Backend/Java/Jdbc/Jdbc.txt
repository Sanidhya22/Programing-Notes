   What do you mean by JDBC ?
-> Java Database Connectivity (JDBC) is an application programming interface (API) for the Java 
   programming language that allows Java programs to access database management systems.

   Java APP  ------>   JDBC  -------->  Database

   How does JDBC works ( Architecture of JDBC ) ?
->         JDBC Application layer         <------------>   JDBC Driver Layer


        Java Application <-->  JDBC API                 JDBC Database Drivers     Databases       
                                                         Oracle Driver      <-->    Oracle    
                                                         SQL Driver         <-->    SQL 
                                                         MariaDB Driver     <-->    MariaDB
