   How we can Display Emoji in HTML Pages ?
-> Emojis look like images, or icons, but they are not.
   They are letters (characters) from the UTF-8 (Unicode) character set. 
   <meta charset="UTF-8">     
   Many UTF-8 characters cannot be typed on a keyboard, but they can always be displayed using
   numbers (called entity numbers)
   <p>I will display &#65; &#66; &#67;</p>

   1. Appends the form data in the url in the key-value pairs.

   Post -
   1. Appends the form data in the body of the HTTP request.
   2. Post method can send unlimited amount of data.

    What do you mean by Web Workers in HTML ?
->  Web worker is a Javascript running in the background without affecting the performance of the page.  
    When executing scripts in an HTML page, the page becomes unresponsive until the script is finished.
    A web worker is a JavaScript that runs in the background, independently of other scripts, without 
    affecting the performance of the page. You can continue to do whatever you want: clicking, selecting 
    things, etc., while the web worker runs in the background.

   What do you mean by Web Accessibility ?
-> Web accessibility is about creating web content, design, and tools that can be used by everyone regardless of
   ability.   

   What are the different ways we can achieve web Accessibility ?
-> The following are the techniques using which we can achieve web accessibility- 

   1. Keyboard Navigation: Ensure all functionality on the website is accessible via keyboard alone. 
   2. Color Contrast: Ensure sufficient color contrast between text and background elements to make content 
      readable for users with low vision or color blindness.
   3. Semantic HTML: Use HTML elements appropriately to convey the structure and meaning of content.    
   4. Alternative Text for Images: Use descriptive alt text (alt attribute) for images to provide context and 
      describe their purpose for users who cannot see them.
   5. Focus Management: Ensure that focus indicators are clearly visible and distinguishable.

   What is the role of semantic tags ?
-> The semantic tags is useful for the following scenarios -
   1. Search Engine Optimization (SEO): Search engines use semantic tags to better understand the content and 
      context of a webpage. By using appropriate semantic tags, developers can improve the visibility and 
      ranking of their webpages in search engine results pages (SERPs).  
   2. Accessibility: Semantic tags improve the accessibility of web content by providing screen readers and 
      other assistive technologies with meaningful information about the structure and purpose of different 
      parts of the webpage.
